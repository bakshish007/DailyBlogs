JavaScript Basics

What is JavaScript?

JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and much more.

Adding JavaScript to Your Web Page

You can add JavaScript to your HTML file by using the <script> tag. Here's an example:

<script>
alert('Hello, World!');
</script>

Variables

Variables are containers for storing data values. In JavaScript, you can declare a variable using var, let, or const.

var name = 'John';
let age = 30;
const isStudent = true;

Data Types

JavaScript supports various data types, including:

  • Strings: 'Hello'
  • Numbers: 10, 10.5
  • Booleans: true, false
  • Objects: { name: 'John', age: 30 }
  • Arrays: [1, 2, 3, 4, 5]

Functions

Functions are blocks of code designed to perform a particular task. Here's an example of a simple function:

function greet() {
  console.log('Hello, World!');
}
greet();

Control Structures

JavaScript supports various control structures, such as:

  • If Statements: if (condition) { // code }
  • For Loops: for (let i = 0; i < 5; i++) { // code }
  • While Loops: while (condition) { // code }

Events

JavaScript can respond to user interactions by handling events. For example, you can add a click event listener to a button:

document.getElementById('myButton').addEventListener('click', function() {
  alert('Button clicked!');
});
