HTML and CSS

HTML Topics

1. Semantic HTML

Semantic HTML refers to the use of HTML tags that provide meaning to the content of a web page. These tags not only help with the structure of the page but also improve accessibility and SEO. Common semantic tags include:

  • <header>: Represents introductory content or a set of navigational links.
  • <nav>: Represents a section of a page that links to other pages or to parts within the page.
  • <article>: Represents a self-contained composition that could be distributed independently.
  • <section>: Represents a generic section of a document.
  • <aside>: Represents content indirectly related to the main content.
  • <footer>: Represents the footer for its nearest sectioning content or sectioning root element.

2. Forms and Input Types

Forms are a crucial part of web development, allowing users to submit data. HTML5 introduced several new input types that enhance form functionality and user experience:

  • <input type="email">: Validates the input as an email address.
  • <input type="tel">: Validates the input as a telephone number.
  • <input type="url">: Validates the input as a URL.
  • <input type="number">: Allows only numerical input.
  • <input type="range">: Provides a slider control for selecting a value within a specified range.
  • <input type="date">: Provides a date picker.
  • <input type="color">: Provides a color picker.

3. Multimedia Elements

HTML5 introduced new elements for embedding multimedia content:

  • <video>: Embeds a video player.
  • <audio>: Embeds an audio player.
  • <source>: Specifies multiple media resources for media elements (<video> and <audio>).
  • <track>: Specifies text tracks for media elements.
<video controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>

CSS Topics

1. Flexbox

Flexbox is a CSS layout module that provides a more efficient way to lay out, align, and distribute space among items in a container. It is particularly useful for responsive design. Key properties include:

  • display: flex;: Defines a flex container.
  • flex-direction: Defines the direction of the flex items (row, row-reverse, column, column-reverse).
  • justify-content: Aligns flex items along the main axis (flex-start, flex-end, center, space-between, space-around).
  • align-items: Aligns flex items along the cross axis (flex-start, flex-end, center, stretch, baseline).
  • flex-wrap: Controls whether the flex items should wrap or not (nowrap, wrap, wrap-reverse).
.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}

.item {
  flex: 1 1 auto;
}

2. Grid Layout

CSS Grid Layout is a two-dimensional layout system for the web. It allows developers to create complex layouts with ease. Key properties include:

  • display: grid;: Defines a grid container.
  • grid-template-columns: Defines the columns of the grid.
  • grid-template-rows: Defines the rows of the grid.
  • grid-column: Specifies the column start and end lines.
  • grid-row: Specifies the row start and end lines.
  • gap: Defines the gap between grid items.
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
  gap: 10px;
}

.item {
  background-color: #f0f0f0;
  padding: 20px;
  border: 1px solid #ddd;
}

3. Animations and Transitions

CSS allows for the creation of animations and transitions to enhance the user experience. Key properties include:

  • transition: Defines the transition effect for changing properties.
  • animation: Defines keyframe animations.
.button {
  background-color: #007bff;
  color: #fff;
  padding: 10px 20px;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease;
}

.button:hover {
  background-color: #0056b3;
}

@keyframes slidein {
  from {
    transform: translateX(-100%);
  }
  to {
    transform: translateX(0);
  }
}

.box {
  width: 100px;
  height: 100px;
  background-color: #007bff;
  animation: slidein 1s ease-in-out;
}
